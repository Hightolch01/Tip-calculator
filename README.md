# Pyside6 Tip-calculator
Проект по теме калькулятор чаевых 
## Структура проекта
main.py - точка входа, настройка "QApplications"  
destipcal.py - интерфейс, сигналы, обработка событий.
## Запуск проекта
**создайте виртуальное окружение:** 'python -m venv venv'  
**активируйте для:**  
**Windows:** "venv\Scripts\activate"  
**macOS/Linux:** "source venv/bin/activate"  
**Установите зависимости:** "pip install pyside6"  
**Убедитесь в включении виртуального пространства и запустите основной скрипт** "python main.py"  
## Описание элементов управления
*Поле "Сумма счета"*	Ввод общей стоимости заказа до чаевых (например, 2500).  
*Поле "чаевые"*	Выбор желаемого процента (стандартное сначение 15%).  
*Поле "Гости"*	На сколько человек нужно разделить итоговую сумму.
*Поле "налог"* Ввод налога в процентах(если он имеется)
*Кнопка "Рассчитать"*	Выполняет математические операции и обновляет результат.  
*Кнопка "Сбросить"*	Сбрасывает все поля для нового расчета.  
*Блок результатов* общую сумму со счетом и долю каждого человека.  
## пример использования
1. Вводим в ячейку справа от поля сумма счета нашу сумму за заказ. К примеру 1000,05
2. К примеру вы хотите оставить чаевые в размере 15%(стандарт)
3. В поле количество гостей выберите 2
4. В ячейке справа от налоги введите 7,5
5. Нажмите кнопку рассчитать
6. В поле вывода покажет:  
   **Итого на человека = 618,16**  
   **Общая сумма = 1236,31**
